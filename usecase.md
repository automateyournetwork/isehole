ISEHole - Transform Cisco Identity Services Engine APIs into Business Ready Documents
=====================================

# ISEHole

ISEHole uses various Cisco Identity Services Engine APIs and transforms them, using Jinja2 templates into:
RAW JSON files
RAW YAML files
CSV files
Markdown files
Dynamic HTML pages
Mindmaps
